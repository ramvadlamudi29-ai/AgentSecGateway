# Follow-up Notes

## Follow-up rule

Follow up every 2-3 days. Stop after 4 messages unless the prospect replies.

## If they ask for demo

Send:

- GitHub repo
- GitHub Pages landing page
- Example report
- Offer 15-minute call

## If they ask for pricing

Send:

- Starter: $299
- Team: $999
- Enterprise: $3k+

## If they ask for private repo scan

Say:

We can run locally or in your private CI. Private code does not need to leave your environment.

## If they ask for compliance

Say:

The audit maps findings to agent security, MCP permissions, secret handling, prompt injection defense, and CI/CD controls.
